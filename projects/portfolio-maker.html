HMS Portfolio Maker

Công cụ tạo portfolio nhanh chóng cho developer chỉ với một file JSON cấu hình đơn giản.

Live Demo Github

Vấn đề gặp phải

Các lập trình viên thường mất quá nhiều thời gian để code lại một trang web cá nhân từ đầu. Mỗi lần muốn cập nhật thêm 1 dự án mới, họ lại phải mò mẫm mở file code ra sửa rất tốn công.

Giải pháp

HMS Portfolio Maker ra đời. Bạn chỉ cần sửa thông tin trong 1 file data.json duy nhất, toàn bộ giao diện trang web sẽ tự động cập nhật lại mà không cần đụng đến code HTML/CSS.

  • Tốc độ cao: Sử dụng Next.js để tối ưu SEO và load trang cực nhanh.
  • Giao diện đẹp: Tích hợp sẵn Tailwind CSS với Dark/Light mode.
